# OTT NFT collection model

## Access

### Genesis Access Pass

- Serial range: #001–#500
- Limited founding edition
- Existing scarcity remains unchanged
- Transferability and access policy remain subject to the validated issuance workflow

### Public Access Pass

- Serial capacity: #000001–#100000
- Wallet-neutral public onboarding edition
- Separate collection identity and metadata from Genesis

## Academy credentials

Academy credentials are designed as proof-of-completion credentials, not speculative art collections.

- XRPL Foundation: #00001–#50000
- Wallet Foundation: #000001–#100000
- Wallet Security: #000001–#100000
- Wallet Operations: #000001–#100000

## Issuance requirements

1. The OTT account completed the required lessons and assessment.
2. The target XRPL wallet ownership is verified by a supported connector.
3. The serial is reserved atomically and cannot be duplicated.
4. Metadata matches course, language, version and learner proof.
5. The mint transaction is signed, submitted and found in a validated ledger.
6. The final NFTokenID and transaction hash are stored with the credential record.

A UI click alone never counts as issuance.
