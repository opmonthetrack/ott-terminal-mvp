# XRPL wallet support matrix

| Wallet/provider | Profile | Direct connect | Transaction signing | Current OTT state |
|---|---:|---:|---:|---|
| Xaman | Yes | Yes | Existing Xaman payload route | Live |
| CROSSMARK | Yes | Official SDK | Official SDK | Beta integration |
| GemWallet | Yes | Official provider API | Official provider API | Beta integration |
| WalletConnect | Yes | Requires Reown project ID | Wallet-dependent | Planned |
| Joey Wallet | Yes | Through WalletConnect | Through WalletConnect | Planned |
| Katz Wallet | Yes | Through WalletConnect | Through WalletConnect | Planned |
| MetaMask XRPL Snap | Yes | Snap RPC | Snap RPC | Planned |
| Ledger | Yes | Reviewed hardware transport required | Device confirmation required | Planned |
| Any classic address | Yes | No | No | Live read-only |

A visible logo never counts as a working connector. OTT marks a connector live only after installation detection, account connection, network detection, signed Testnet transaction, validated result and disconnect/session recovery have all been tested.
