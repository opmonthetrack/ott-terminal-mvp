learn first

## What The MVP Shows

### 1. Dashboard

Shows the main project story, SourceTag and reward/proof overview.

### 2. Partner Hub

Shows education-first partner routes with risk notes, checklists and optional Proof Stamp flow.

### 3. XRPL Verify

Shows how transaction hashes can be checked for SourceTag and proof logic.

### 4. Reward Ledger

Shows local XP and reward events. XP is off-chain and has no guaranteed financial value.

### 5. Truth Desk

Shows a paid education/service route for short questions or 1-on-1 feedback.

### 6. Access Gate

Shows access route concepts: fiat provider route, XRP payment, RLUSD concept and NFT Access Pass concept.

### 7. Pitch Mode

Shows the 2-minute pitch flow for the project.

### 8. Submission Pack

Shows copy-ready submission text and demo order.

## Compliance-Safe Position

The Terminal is education-first.

It does not:

```txt
custody funds
act as broker
provide yield
execute trades
promise token value
promise investment returns
```

It does:

```txt
explain routes
show risk awareness
route users to official providers
create Xaman payloads
verify XRPL proof
track local education progress
```

## Why XRPL

XRPL is fast, low-cost and suitable for payments, issued assets, memos, transaction verification and proof-based user routes.

## Why SourceTag

The SourceTag gives the project a recognizable proof identity on XRPL.

```txt
2606170002
```

It allows the app to verify that a transaction belongs to the correct route or proof flow.

## Best Judge Question Answer

**Question:** Is this an exchange, wallet or DeFi product?

**Answer:**

No. The Terminal is an education and proof layer. It does not custody funds, does not act as broker, does not provide yield and does not execute trades. It helps users understand routes and verify meaningful XRPL actions.

## Closing Line

This is not another wallet dashboard. It is a guided XRPL education and proof layer built to help users, builders and partners move safely on the track.
